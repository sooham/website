# Website
Built using:
- React.js
- Webpack and its infinite plugins and loaders
- CSS modules!

Written in ES6 JS.

## Installation
At the root of this repository.
```bash
npm i -g webpack webpack-dev-server
npm i
NODE_ENV=production npm start
```
Then navigate to `localhost:8080`
